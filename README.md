# Courtly

Courtly is a Flutter iOS app built around reception planning, guest courtesies,
and occasion readiness. The current app shell uses Cupertino navigation with
three working sections:

- Docket: reception timing, preparation threads, and protocol cues.
- Guests: personal welcome notes, seating texture, and care boundaries.
- Wardrobe: readiness signals for attire, weather, and accessory choices.

The Dart code is organized by business area under `lib/features`, with shared
presentation pieces in `lib/shared` and app-level navigation/theme files in
`lib/atelier`.
